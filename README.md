#  Fire & Smoke Detection Using YOLOv8

##  Overview

This project implements a deep learning-based Fire and Smoke Detection system using **YOLOv8 (PyTorch)**.  
The trained model detects fire and smoke in images and can be applied in safety monitoring and surveillance systems.

---

##  Model Details

- **Architecture:** YOLOv8 Nano  
- **Framework:** PyTorch  
- **Trained Weights:** `best.pt`  
- **Dataset Format:** YOLO (train / valid / test)
